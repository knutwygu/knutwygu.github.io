Spatial vs. Temporal Edge Detection

Signal Processing Computer Vision Python

A comparative study of two Linear Time-Invariant (LTI) systems performing edge detection in different domains, bridging abstract DSP theory with modern event-based vision technology.

  • System A (Spatial Filtering): Implemented 2D convolution using Sobel operators. Extracted all high-frequency spatial components, including redundant background textures.
  • System B (Temporal Filtering): Simulated an Event Camera (DVS) using a discrete-time difference equation. Acted as a temporal high-pass filter, successfully suppressing static backgrounds and isolating moving objects.
  • Core Finding: Demonstrated that temporal edge detection is significantly more efficient for motion-tracking tasks by producing a sparser signal and reducing redundant data processing.

Project_00 2026

Music Reactive LED System

Mixed-Signal Analog Filters Embedded C++

This project aims to design and build a real-time music reactive LED system that combines analog audio circuits with embedded digital control. It was developed as a mixed-signal design project at UC Irvine.

  • Analog Front-End: Features a custom preamplifier circuit designed using an LM358 operational amplifier to process millivolt-level electret condenser microphone outputs.
  • Hardware Frequency Separation: Implements a 3-band active analog filter network to cleanly separate the audio spectrum into bass, midrange, and treble components before digitization.
  • Embedded Control: Uses an Arduino Mega running non-blocking embedded C/C++ firmware to sample signals and map them to a WS2812B RGB LED strip in real-time.
  • Interactive UI: Includes a 0.96-inch OLED display and a rotary encoder allowing users to switch between Spectrum Mode, Waveform Mode, and Reactive Strobe Mode.
